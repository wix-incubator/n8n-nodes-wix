#### 1.0.0

Initial release of the n8n community node for Wix.

**Features:**

- **Wix API Credentials**: OAuth2 authentication support for connecting to Wix sites
- **Products Resource**: Full CRUD operations for Wix Store products
  - Create, read, update, and delete products
  - Support for product variants, inventory, media, and custom fields
- **Orders Resource**: Manage Wix Store orders
  - Create, read, update, and cancel orders
  - Query and filter orders
- **Wix Automations Trigger**: Listen for Wix automation events via webhooks
- **List Search**: Dynamic dropdowns for selecting products and orders in the n8n UI
